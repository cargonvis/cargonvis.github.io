B.Eng. in Data Engineering

Duration: August 2020 - August 2023

Institution: Satakunta University of Applied Sciences (SAMK), Finland

Field of Study: Data Engineering

Specialization: Artificial Intelligence

Description:

When I started my journey at SAMK, I had no prior experience in programming, but I quickly fell in love with AI and robotics. My Bachelor of Engineering in Data Engineering with a specialization in Artificial Intelligence provided me with a comprehensive foundation.

The curriculum included foundational courses like Basic Developer Tools and Version Control (UNIX commands and basics of Git), GIT Open, Linux Basics, and Container Technologies (using Docker). I progressed through programming courses such as Programming Basics (using Python), Python Programming, AI Programming (also using Python), Object-Oriented Programming (using C#), C++ Programming, C Programming, Embedded C/C++ Programming, and an IoT course.

In web development, networks, and cloud computing, I took courses like Introduction to Networks, Web Development Basics (HTML and CSS), and Cloud Computing and GPUs (using Azure and Google Colab). My database courses included Relational Databases, Database Programming, Big Data, and NoSQL Databases. I worked with MariaDB, MySQL, SQLite, MongoDB, Redis, and Neo4j.

I gained extensive knowledge in data analysis, covering data acquisition, cleaning, preprocessing, and analysis using AI tools. This solidified my ability to efficiently harness AI for practical data solutions.

In AI, I studied Machine Learning, Neural Network Models, Machine Vision and Convolutional Neural Networks, and Natural Language Processing (NLP). These courses deepened my understanding of AI, enabling me to develop sophisticated models and applications.

Robotics was a significant part of my studies, with courses such as Robotics, Open Source Robot Platforms (ROS), Mechatronics and Robot Programming, and participation in an international week in Belgium focused on programming drones using ROS.

My bachelor's thesis, titled "Deep Learning EEG-based Motor Imagery System for Robot Control using 3D Printed Headset and Electrodes", showcases my commitment to merging AI with practical applications. The project involved designing and implementing an EEG-based motor imagery system to control a robot arm using a 3D printed headset and electrodes. This research demonstrated the potential of custom EEG headsets and deep learning models in creating reliable and user-friendly robot control systems.

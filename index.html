ROS Project with Object Detection

Development of a Object Detection model that detects tyres stacks from a pallet and is able to send the information to a robot (coordinates, number of stacks, number of tyres per stack). Use of ROS for connecting depth cameras and recording rosbags from the environment. Those rosbags were transformed into video and then into frames for training the object detection model. Image annotation with labelImg and training an object detection model with YOLOR. Use of python.

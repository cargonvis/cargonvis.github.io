TrainScanner Project with Object Detection

Company: Vire labs Oy     Working time: November 2021 - April 2022

 

Development of an image detection system that is able to analyze scanned trains images (raw data) and identify each unit (wagon) of it. Annotating images by using labelImg and training of a deep neural network. The images are processed with YOLOv4 and Darknet locally by using "Virebox". Use of Python.
